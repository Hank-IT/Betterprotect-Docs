Betterprotect includes a basic permissions system.

The following roles are available:

## Read Only
The user may only perform read operations for the main menu and the policy.

## Authorizer 
The user may create new Whitelist and Blacklist entries and install the policy. Read Only role is included.

## Editor 
The user may perform write operations for the main menu and the policy. Read Only and Authorizer role is included.

## Administrator
The user may perform all operations on the main menu, the policy and the system menu. All other roles are included.
